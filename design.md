# Early Bird Grand Prix - Design Document

## App Concept
マリオカート風のポップでエネルギッシュなUIを持つ、レース型早起き習慣化アプリ。
「昨日の自分（ゴースト）」と毎日レースし、1分ずつ起床時間を早めていく。

## Screen List

### 1. Home (Race Dashboard)
- メインのレース画面。カートがコース上を走るビジュアル
- 今日の目標起床時間を大きく表示
- 「起床！」ボタン（大きく目立つCTAボタン）
- 現在のストリーク（連続起床日数）
- 今日の勝敗ステータス（勝利/敗北/未判定）
- コイン残高表示
- 現在のコース名とランク

### 2. Growth Chart (成長チャート)
- FXのローソク足/ラインチャート風の成長曲線
- 累計成長率（1.001^n）のグラフ
- 起床時間の推移グラフ
- 日別の勝敗履歴（カレンダービュー的な表示）

### 3. Course & Items (コース＆アイテム)
- 現在のコースとプログレスバー（次のコースまでのコイン進捗）
- 全10コースのリスト（アンロック状態表示）
- 所持アイテム一覧（キノコ、スター、キラー）
- アイテム使用ボタン
- カートの見た目（キラーによるアップグレード段階）

### 4. Night Routine (ナイトルーティン)
- 推奨就寝時間の表示（起床時間から逆算）
- 「ピットイン（就寝）」ボタン
- 明日の目標起床時間
- 就寝前チェックリスト的なUI

### 5. Settings (設定)
- 初期起床時間の設定
- 最終目標時間の設定
- 睡眠時間の設定（推奨就寝時間計算用）
- 就寝リマインダー時間の設定
- データリセット

## Primary Content and Functionality

### Home Screen
- **カートレースビジュアル**: コース上を走るカートのイラスト（SVGベース）
- **目標時間表示**: 今日の起床目標時間（大きなデジタル時計風）
- **起床ボタン**: タップで起床記録。成功時にエフェクト表示
- **ゴースト比較**: 昨日の起床時間との比較表示
- **コイン表示**: ヘッダーにコイン数をマリオ風に表示

### Growth Chart
- **ラインチャート**: 成長率の推移（1.001^n）
- **起床時間チャート**: 目標 vs 実際の起床時間
- **統計サマリー**: 総成功日数、現在の成長率、最長ストリーク

### Course & Items
- **コースマップ**: 10コースを縦に並べたロードマップ
- **プログレスバー**: 現在コースの進捗
- **アイテムカード**: 所持アイテムをカード形式で表示

## Key User Flows

### 朝の起床フロー
1. アラームで起床 → アプリを開く
2. Home画面で「起床！」ボタンをタップ
3. 判定: 目標時間内なら勝利エフェクト、超過なら敗北表示
4. コイン獲得アニメーション
5. 翌日の目標時間が更新される

### 夜の就寝フロー
1. Night Routineタブを開く
2. 推奨就寝時間を確認
3. 「ピットイン」ボタンをタップ
4. 就寝記録完了 → ボーナスコイン獲得

### アイテム使用フロー
1. Course & Itemsタブを開く
2. 所持アイテムから使用したいものを選択
3. 確認ダイアログ → 使用
4. 翌日に効果が適用される

## Tab Bar Structure
1. **Home** (house.fill) - レースダッシュボード
2. **Chart** (chart.line.uptrend.xyaxis) - 成長チャート
3. **Course** (flag.checkered) - コース＆アイテム
4. **Night** (moon.fill) - ナイトルーティン
5. **Settings** (gearshape.fill) - 設定

## Design Principles
- **ダークテーマベース**: 夜空をイメージした深いブルーの背景
- **ネオン風アクセント**: ボタンやハイライトにネオン風のグロー効果
- **レトロゲーム感**: ピクセル風フォントやドット絵的な装飾
- **エネルギッシュ**: 鮮やかな色使いとダイナミックなレイアウト
- **ワンハンド操作**: 重要なボタンは画面下部に配置
